# Тестовое задание: CRUD приложение на React

Демонстрация: https://users-list-drab.vercel.app

## Клонирование репозитория

Склонируйте репозиторий к себе на компьютер:

```bash
git clone <URL_репозитория>
```

Перейдите в папку проекта:

```bash
cd <имя_папки_проекта>
```

## Установка зависимостей

Установите необходимые пакеты:

```bash
npm i
```

## Переменные окружения

1. Скопируйте `.env.example` в `.env.local`:

```bash
cp .env.example .env.local
```

2. Заполните значения переменных в .env.local, например:

```bash
NEXT_PUBLIC_API_URL=https://<ваш-проект>.mockapi.io
```

## Запуск проекта

Запустите проект в режиме разработки:

```bash
npm run dev
```

Откройте http://localhost:3000 в браузере.

## Пояснительная записка

Приложение позволяет управлять списком пользователей:
- CRUD операции: создание, чтение, редактирование и удаление пользователей;
- Таблица с колонками: ID, Имя, Email, Телефон, Роль;
- Все компоненты и логика написаны с использованием TypeScript;
- Валидация форм с помощью Zod и React Hook Form;
- UI реализован на Ant Design;
- Состояние управляется через Redux Toolkit (взаимодействие с api через axios);
- Для хранилища используется [mockAPI](https://mockapi.io/)
- Выполнен деплой на Vercel https://users-list-drab.vercel.app
- Реализация выполнена строго в соответствии с требованиями ТЗ; дополнительные функции не добавлялись.
